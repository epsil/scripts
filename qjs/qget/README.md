`qget`
======

**`qget`** er en wrapper for `youtube-dl`. Den lagrer metadata som YAML.

Installasjon
------------

Kjør:

    cd qget
    npm install
    sudo npm link

Dette gjør kommandoen `qget` tilgjengelig fra kommandolinjen (for hjelp, skriv `qget --help`).
