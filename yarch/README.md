`yarch`
=======

**`yarch`** er en wrapper for `youtube-dl`. Den lagrer metadata som YAML.

Installasjon
------------

Kjør:

    cd yarch
    npm install
    sudo npm link

Dette gjør kommandoen `yarch` tilgjengelig fra kommandolinjen (for hjelp, skriv `yarch --help`).
